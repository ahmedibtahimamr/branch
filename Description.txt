
## Description

- `index.html`: The main HTML file that includes the SVG clock and links to the CSS and JavaScript files.
- `style.css`: The CSS file that styles the clock.
- `script.js`: The JavaScript file that updates the clock hands every second.
- `README.md`: This file providing an overview of the project.
